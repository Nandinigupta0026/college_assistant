# 🎓 College Assistant

AI-powered personal assistant for college students built with Amazon Nova.

## Features
- Auto-detects deadlines from Telegram groups
-  Saves to Google Calendar automatically  
-  Smart message filter
-  Daily morning summary at 8am

## Tech Stack
- Amazon Nova (AWS Bedrock)
- Google Calendar API
- Telegram Bot API
- Python + FastAPI



## Setup
1. Clone the repo
2. Install requirements: `pip install -r requirements.txt`
3. Add your API keys to `.env`
4. Run: `python telegram_reader.py`
